解锁网易云灰色歌曲

解锁网易云灰色歌曲

UnblockNeteaseMusic是一款可以给你解除网易云音乐海外限制并解锁变灰(无版权)歌曲的工具，让你可以听取客户端任何的歌曲，而不受到限制

灰色解锁歌曲来源于？

网易云旧链/QQ/虾米/百度/酷狗/酷我/咕咪/JOOX音源替换变灰歌曲链接(默认仅启用前四

支持的设备类型？

Windows、MacOS、安卓、IOS等设备

UnblockNeteaseMusic项目地址？

https://github.com/nondanee/UnblockNeteaseMusic

需要准备的东西？

一台有公网IP地址的服务器，Windows网易云客户端、Android网易云客户端

因为本人是穷逼，所以mac跟ios无法进行测试

搭建步骤

1、连接到你的服务器，这里我用的是白嫖一年的谷歌云

# CentOS系统 首先获取管理员权限
sudo -i
# 安装nodejs
curl -sL https://rpm.nodesource.com/setup_10.x | bash -
sudo yum -y install nodejs

2、运行UnblockNeteaseMusic

# 从git地址克隆该项目
git clone https://github.com/nondanee/UnblockNeteaseMusic.git
# 进入到指定目录下
cd UnblockNeteaseMusic
# 启动 默认端口是8080 如果需要修改可追加-p 端口号 
# 例如：node app.js -p 4567 则启动端口为4567
node app.js

Windows客户端设置

设置成功后，可以点击测试查看代理地址是否可用，之后会提示重启，就能听歌曲啦

这里测试用的是周杰伦等你下课，没原因，单纯就是非常喜欢这首歌

Android客户端设置

保存后关掉WLAN跟网易云客户端，重新连接一下试试看~

这里选了另外一首周杰伦非常耐撕的一首歌曲，烟花易冷
